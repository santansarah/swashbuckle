Getting Started with QAComplete - User Tutorial

This tutorial is for novice QAComplete users. It will help you get started with managing requirements, tests, and defects in QAComplete.

Before you start working with the tutorial, please make sure that you have a QAComplete account, and that account has access to a project with which you can experiment. We recommend that you use the Sample Project that comes with QAComplete Trial.

To learn how to create a project and how to configure accounts, see Getting Started with QAComplete - Administrator Tutorial.

This tutorial includes the following steps:

Requirements

1. Log in to QAComplete

2. Select a Project

3. Create a Release

4. Create a Requirement

5. Create Tests and a Test Set

6. Run Tests

7. Create a Defect (Bug)

8. View Linked Items

9. Reports and Dashboards
